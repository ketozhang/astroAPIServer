Global Query Parameters

/api/path/to/endpoint?{param1}={value1}&{param2}={value2}

Parameter Description
select Limit the output to only a certain set of columns.
limit Limit the output to only a certain set of rows.
orderby Order the output by a column or ordered set of columns.
desc Order of the output is descending by default. Set to False for ascending order.
format (required) Output can be formatted as either JSON (default), ASCII, or CSV.
